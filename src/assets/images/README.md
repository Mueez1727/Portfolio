# Images Folder

Place your profile image here as `hero-profile.png`.

## Usage

Import in your components like this:

```jsx
import heroImage from '../assets/images/hero-profile.png';

// Then use in JSX:
<img src={heroImage} alt="Profile" />
```

## Recommended Specifications

- **Format:** PNG or JPG
- **Size:** 400x500 pixels (or similar aspect ratio)
- **Quality:** High resolution for retina displays
